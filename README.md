# Confidence Portfolio

A personal portfolio website built with HTML and CSS to showcase my projects, skills, and web development journey.

## About This Project

This portfolio was designed and developed by me as the Capstone Project for Dr. Angela Yu's Complete Web Development Bootcamp. The goal was to create a fully responsive personal website that highlights my skills, projects, and experience as an aspiring full-stack web developer.

## Features

* Responsive multi-page website
* Home page with personal introduction
* About Me section
* Skills section
* Projects showcase
* Contact page
* GitHub and LinkedIn links
* Mobile-friendly design

## Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Grid
* Bootstrap
* Git
* GitHub
